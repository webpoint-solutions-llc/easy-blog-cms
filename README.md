# Easy blog CMS

## Installing / Getting started

1. Clone the project
   `git clone `

2. Run project with docker

   `docker-compose up -d`

3. To seed database/migrate
   `docker-compose exec service yarn migrate`

### or

## Using make

1. Clone the project
   `git clone `

2. Run project with docker

   or
   `make up`
   `make build`

3. To seed database/migrate
   `make migrate`
